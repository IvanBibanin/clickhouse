# Запуск
В docker-compose.yml в environment вы можете задать свои параметры для входа\
По умолчанию они равны:\
Имя базы: raw_layer\
Юзер: admin\
Пароль: admin\
Порт:8123\
# user_files
Файлы из папки user_files на вашем компьютере будут доступны внутри контейнера\
ClickHouse сможет читать файлы из этой директории (например, для импорта данных)\
Вы можете легко добавлять/удалять файлы без перезапуска контейнера\
Изменения с обеих сторон синхронизируются в реальном времени\
В папке находится словарь ru.bin для работы с функцией lemmatize
